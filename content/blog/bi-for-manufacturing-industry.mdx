The project involves building a data warehouse with facts and dimensions for the manufacturing industry, using SQL scripts to populate the database structure. Business queries and drill-down capabilities were created to extract insights, and around 20 dashboards were developed in Tableau. These dashboards allow dynamic input and display of data, enabling users to drill down into specific details. Predictive analytics models were also built to simulate changes in sales revenue based on factors like discounts or an increased sales force. The deployment of these dashboards provides valuable insights to the manufacturing sector.
